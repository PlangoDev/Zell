# BrainFormer and Zell — Documentation Set

BrainFormer is a brain-inspired language model trained without backpropagation. It composes neuroscience-derived mechanisms — cerebellar Marr-Albus expansion coding, sparse distributed representations, k-winners-take-all selection, local learning rules, multi-timescale context binding, retrieval/associative memory, and test-time plasticity — into a deep stack of independent cerebellar layers combined by a learned probabilistic mixture, implemented as a single C/CUDA translation unit (`test_013_v3/src/main.cu`) for the Kaggle dual NVIDIA T4 (`sm_75`) target. Every learned tensor is updated by a rule that reads only quantities local to the point of update: granule weights by competitive k-means under winner-take-all, the hierarchical class/word readout by a local delta rule, and the layer mixture by a local EM step. The reference baseline is EleutherAI/Pythia-410M (405,334,016 parameters) at WikiText-103 strided perplexity 17.19 (window 512, measured by `run_llm.py`). The program established that the perplexity gap from BrainFormer's operating range to that baseline is structural. Zell is the hybrid that follows from that finding: a backprop-trained efficient transformer (full fine-tune of `Qwen/Qwen2.5-0.5B-Instruct`) supplies the coherence core, and the two BrainFormer mechanisms that produced realized wins — infini-gram retrieval and test-time adaptation — are retained as inference-time augmentations.

## Status

The no-backprop model is a working, fast language model that sits roughly two orders of magnitude above the 17.19 baseline in static perplexity; the gap is structural — the parallel-expert mixture with fixed random codes and a linear local-delta readout lacks compositional credit assignment, and no surveyed data-side or grafted technique crosses it, so the realistic ceiling for the pure model is low hundreds of perplexity on in-distribution text rather than coherent open-ended generation. Within that boundary the two augmentations are real and large but in-distribution only: test-time adaptation cut the 4-layer multiscale static perplexity from 3412 to 1930 (-43%) on the document being read, and infini-gram interpolation (50M index, lambda 0.3) cut it from 3412 to 1272 static (-63%) and 1930 to 762 adaptive (-60%), the single largest realized quality jump — both falling back to the bare model on novel chat input where no long suffix matches and there is nothing to fixate on. Because the structural gap cannot be closed by the no-backprop design, the Zell hybrid moves the coherence burden to a backprop-trained efficient transformer core and keeps infini-gram retrieval and test-time adaptation as inference-time augmentations, with the same honest scope carried forward.
